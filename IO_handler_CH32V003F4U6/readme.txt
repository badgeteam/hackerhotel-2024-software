This project was originally a WCH Eclipse sample project which we've slightly
modified to fit the typical Embeetle project structure.

The original resources can be found on:
http://www.wch-ic.com/products/CH32V003.html

After unzipping this ZIP-folder, the general source code is copied from
'CH32V003EVT/EVT/EXAM/SRC/' while the GPIO toggle code is copied from
'CH32V003EVT/EVT/EXAM/GPIO/GPIO_Toggle/User/'. The linkerscript is found at
'CH32V003EVT/EVT/EXAM/SRC/Ld/Link.ld'.
